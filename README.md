## Installation

Set a virtual environment after clone the project
```sh
python3 -m venv venv
```

Activate the virtual environment
```sh
source venv/bin/activate
```

Install the dependencies.

```sh
pip install -r requirements.txt
```

Reun the project

```sh
python3 main.py
```


**Enjoy!**
